# bachelor-global-website-comparison
Dieses Repository enthält die Implementierung und Auswertungswerkzeuge für eine Bachelorarbeit zum Vergleich von Webseiten aus verschiedenen Ländern. Untersucht werden visuelle Gestaltung, Webseitenkomplexität und kulturelle Unterschiede anhand automatisierter Analysen von HTML-Dokumenten, Screenshots und Webmetriken.
